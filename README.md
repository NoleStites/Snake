# Snake: A Minimalist Arcade Experience

This version of Snake is a polished, browser-based tribute to the classic arcade game. It prioritizes responsive controls, clean design, and a customizable visual experience optimized for both desktop and mobile play.

---

### How to Play

**Objective**
Guide the snake to consume apples to grow in length and increase your score. The game ends if the snake collides with the board boundaries or its own body.

**Desktop Controls**
* **W / Arrow Up**: Move Up
* **A / Arrow Left**: Move Left
* **S / Arrow Down**: Move Down
* **D / Arrow Right**: Move Right

**Mobile Controls**
* **Swipe**: Flick your finger in any direction to change the snake's heading.
* **D-pad**: Toggle the on-screen directional buttons via the checkbox beneath the game board for precise tactile control.

**Customization**
Use the theme selector at the top of the menu to switch color palettes to your liking.
